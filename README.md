WebRPG
============

WebRPG ist ein Online Browser Rollenspiel. Welches im Rahmen des Werkmoduls WebAdvancedII entsteht.
Es befindet sich in ständiger Weiterentwicklung und befindet sich derzeit noch in einer frühen Testphase. Story und Umgebung befinden sich noch in Planung. Die wichtigsten Funktionen sind bereits Implementiert und können leicht erweitert werden. 
Um eine Umgebung zu erschaffen ist es z.B. nur nötig weitere map.png's zu laden, der Code muss dafür theoretisch nicht verändert werden.

Zum Einsatz kommen die folgenden Webtechniken:
Processing.js - zur Darstellung der gesamten Spielwelt
node.js - als Server zur Kommunikation unter den Spielern
socket.io - als node.js Plugin zur ständigen Aktualiesierung der anderen Spieler und zum Chatten
MongoDB - zur abspeichern der Spielerposition

Die aktuellste Version kann über http://webrpg.jit.su/ getestet werde.
Derzeit gibt es noch Performance Probleme mit FireFox, bitte Chrome verwenden.
